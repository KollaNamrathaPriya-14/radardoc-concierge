# RadarDoc Concierge

Multi-agent AI for radar technical documentation (Capstone Project for Google 5-Day AI Agents Intensive).

## Features
- Research agent (web search + summarization)
- Math agent (symbolic derivation using SymPy + code execution tests)
- Diagram agent (auto-generated block diagrams)
- Writer agent (final PDF compilation)
- Multi-agent orchestration with async + structured pipeline
- Memory Bank for session-based storage
